3. Welche Daten wir erheben

Diese Website ist ein Ausbildungsprojekt und erhebt keine personenbezogenen Daten. Es gibt keine Kontaktformulare, keine Benutzerregistrierung und keine Zahlungsabwicklung.

3.1 localStorage (Browserspeicher)

Die Anwendung nutzt den localStorage Ihres Browsers, um folgende funktionale Daten lokal auf Ihrem Gerät zu speichern:

  • Warenkorb — ausgewählte Speisen und Mengen
  • Wunschliste — als Favorit markierte Speisen
  • Letzte Bestellungen — zuletzt in den Warenkorb gelegte Speisen
  • Speisekarte — Zustand der Speisekategorien

Diese Daten verlassen niemals Ihren Browser und werden nicht an Server oder Dritte übermittelt. Sie können diese Daten jederzeit über die Browsereinstellungen löschen.

3.2 Technische Daten beim Hosting

Sofern diese Website über einen Webserver bereitgestellt wird (z. B. GitHub Pages), können durch den Hosting-Anbieter automatisch technische Zugriffsdaten erhoben werden:

  • IP-Adresse
  • Browsertyp und -version
  • Betriebssystem
  • Datum, Uhrzeit und Dauer des Zugriffs
  • Aufgerufene Seiten

Diese Daten werden durch den Hosting-Anbieter gemäß dessen eigener Datenschutzerklärung verarbeitet.
